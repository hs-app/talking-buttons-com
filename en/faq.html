FAQ & Support

Where do I install Talking Buttons?

Android: Google Play. Chrome / Chromebook: Chrome Web Store.

How do I move boards between the phone app and the Chrome extension?

Use export in one app to save an .obz (or .obf) file, then import that file in the other. Both follow the Open Board Format.

Text-to-speech sounds wrong or is missing

On Android, install or select the language you need in system Text-to-speech settings. On Chrome, voices depend on the browser and OS; check Chrome’s accessibility / speech settings and installed languages.

Microphone / recording

Recording is optional. Android and Chrome will only use the microphone when you start a recording. If something fails, check app or site permissions in system settings.

Is internet required?

Boards work offline once set up. You need a connection only to install or update the app or extension, or if you use online content outside Talking Buttons.

More help

The Chrome extension includes a built-in user guide in the side panel. For other questions, email support@talking-buttons.com.
