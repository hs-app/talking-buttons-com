# AAC board files (OBF / OBZ)

Place **Open Board Format** files here using the **same filenames** referenced in:

- [`en/boards.html`](../../en/boards.html)
- [`ru/boards.html`](../../ru/boards.html)

Until a file is present at the expected path, its download link on the site will return 404. Replace or remove rows in those HTML files when you publish real boards.

Supported extensions: **`.obz`** (archive, typical export) or **`.obf`** (single board JSON).
