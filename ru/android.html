Приложение для Android

С помощью мобильного приложения для Android инструменты АДК всегда будут с вами на вашем телефоне или планшете. Для работы в приложении коммуникационные доски не требуют подключения к интернету, все данные хранятся локально на вашем устройстве. Предварительно настроенные доски можно импортировать из нашего расширения для Chrome или других приложений, поддерживающих Open Board Format.
